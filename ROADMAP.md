# Roadmap

1) Foundations
- Repo setup, tooling, lint/format, basic layout shell
- Define visual language and shared components

2) Scenarios & Data
- Write initial scenario fixtures and mock API behaviors
- Configure XState machines for login/refresh/logout with telemetry hooks

3) Visualization Core
- Flow timeline/graph with stepper controls
- Token & cookie inspector panel synchronized to state transitions

4) Edge Case Lab
- Implement presets: expired refresh, clock skew, multi-tab logout, bad callback
- Explanation copy for each transition and failure

5) Polish & Demo
- Demo mode script, seeded walkthroughs, and screenshot scenes
- Accessibility review and documentation pass
